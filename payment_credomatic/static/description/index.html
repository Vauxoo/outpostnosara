HOW CAN I USE IT?

Installation
Configuration
User manual for website
User manual for the backend sales

Installation

Apps > Search Credomatic
app > Install

1

Configuration

Navigate into Website > Configurations > Payment Acquirers..

2

Find Debit / Credit card (That is Credomatic).

3

Once it's opened, publish it. It's necessary for it to appear in the website.

4

Set your Credomatic's credentials.

5

Save the record.

6

User manual for website

Navigate into the Website's shop.

7

Choose a product and add it to your shopping cart.

8

Once it's opened, publish it. It's necessary for it to appear in the website.

9

Once in the payment section, choose "Credit / Debit card" and fill the information to pay.

10

User manual for backend

Navigate to sales.

11

Create a quotation.

12

Click the button "Send by email".

13

Click the button "Preview", so you can open that quotation.

14

Choose "Credit / Debit card" and fill the information and pay the quotation.
